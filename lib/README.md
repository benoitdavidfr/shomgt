## éléments partagés
Ce module contient les divers packages suivants utilisés par les autres modules :

  - coordsys permet d'effectuer des changements simples de système de coordonnées a priori sur l'ellipsoide IAG_GRS_1980,
  - gegeom est un package géométrique utilisant des coordonnées géographiques ou euclidiennes,
  - gebox définit des classes définissant un BBox avec des coord. géographiques ou euclidiennes,
  - zoom définit la classe Zoom regroupant l'intelligence autour des niveaux de zoom,
  - mysql définit les classes MySql et MySqlResult utilisées pour exécuter des requêtes MySql.
